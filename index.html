K T U P AD

MVC Framework
Adalah Framework berbasis Client-Server, dengan JSON sebagai penghubung data.

Mudah, Praktis dan Cepat.
Terprediksi, konsisten dan modular. ini yang membuat ktupad mudah digunakan bagi pengembang dan pengguna.
Responsive. ktupad praktis, cukup satu aplikasi untuk Web App, Dekstop dan Mobile.
Progressive Web Apps (PWA), memungkinkan bekerja secara offline, waktu load data menjadi jauh lebih cepat.

Demo & Download

Model

Adalah PHP Framework dengan fungsi query database, diantaranya Create, Read, Update, Delete (CRUD). juga fungsi Export dan Import CSV.

Hasil query berupa JSON, yang didapat dengan menjalankan perintah menggunakan AJAX.

Bekerja dengan menyertakan ktupad.php pada file modul.

<?php include('ktupad.php');?>

Demo & Download

View

Adalah CSS Framework diantaranya Icons, Form, Table, Grid.

lebih cepat dan praktis, ini dikarenakan penggunaan teknologi Progresive Web Apps (PWA) yang memungkinkan aplikasi bekerja secara Offline .

Bekerja dengan menyertakan ktupad.css pada file modul.

<link rel="stylesheet" type="text/css" href="style.css">

Demo & Download

Controller

Adalah JS Framework diantaranya Tabledata, Pagination, Notify, Dynamic Form, HTML Editor dan Ajax.

Rewrite dan resource, ini dikarenakan penulisan berbasis pada object.

Bekerja dengan menyertakan ktupad.css pada file modul.

<script type="text/javascript" src="ktupad.js"></script>
